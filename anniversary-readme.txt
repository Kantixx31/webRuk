# Anniversary Website

A special anniversary website created to celebrate 5 years of love and memories. This website features:
- Password-protected access
- Photo gallery with memories
- Countdown timer to the anniversary
- Music player
- Special message page
- Beautiful pink and white theme

## Setup Instructions

1. Clone this repository:
```bash
git clone https://github.com/your-username/anniversary-website.git
```

2. Add your photos to the `images` folder
3. Add your music file to the `audio` folder
4. Customize the message in `index.html`
5. Update the password in `js/script.js` if needed

## Customization

- Default password is set to "123456"
- Anniversary date is set to February 14, 2025
- You can modify the style in `css/style.css`
- Add your personal message in the message section of `index.html`

## Technology Used

- HTML5
- CSS3
- JavaScript
- Font: TCS4Dinner
